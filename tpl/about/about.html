О курсе

Этот курс создан для того, чтобы научить вас основам разработки веб-приложений "с нуля". На данном курсе вас ждёт:

  • Введение в веб (что это такое, на чем он основан, протоколы, какие веб-технологии есть);
  • Всё про клиентскую сторону (что это такое, базовый набор веб-разработчика: HTML, CSS, JS);
  • Базы данных (реляционные БД, SQL, ORM и т.д.);
  • Серверная часть веб-приложений (что это такое, инструменты для написания серверной части: язык python и веб-фрэймворк django);
  • Системы контроля версий : Git;
  • Тестирование;
  • Развертывание приложения;
  • Разработка реального проекта.
