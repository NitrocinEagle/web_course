jQuery

Что такое JQuery? И зачем он нужен

jQuery – это JavaScript-библиотека, фокусирующаяся на взаимодействии JavaScript, HTML и CSS.

  • Обращаться к любому элементу DOM (объектной модели документа) и не только обращаться, но и манипулировать ими.
  • Работать с событиями.
  • Легко осуществлять различные визуальные эффекты.
  • Работать с AJAX (технология, позволяющая общаться с сервером без перезагрузки страницы)
  • Имеет огромное количество JavaScript плагинов, предназначенных для создания элементов пользовательских интерфейсов.

Как начать пользоваться JQuery?

  • Скачать саму библиотеку на сайте разработчиков - https://jquery.com/
  • Разорхивировать её и перенести в ту же папку, где находится наша html-страница
  • Подключить jQuery к html-странице:
    <script type="text/javascript" src="jquery.js"></script>
  • Создать js-файл и тоже подключить его к странице:
    <script type="text/javascript" src="script.js"></script>


Синктасис JQuery

$('селектор').действие('свойства действия');
Cелектор - элемент или элементы, с которыми мы будем что-либо делать.
Действие - что именно мы будем делать с выбранными элементами. Мы можем добавить какой-либо эффект, css-стиль, изменить html-код и т.д. Здесь же могут быть указаны какие-либо события.
Свойства действия - если они предусмотрены действием.


Селекторы

Селекторы - это элементы, с которыми мы хотим что-либо сделать.

  • #id - выбирает единственный элемент с переданным идентификатором (id).
    $("#lok").css("border", "1px solid red");
  • .class - выбирает все элементы с переданным классом.
    $(".lok").css("border", "1px solid red");
  • elements - выбирает все элементы с выбранным именем.
    $("div").css("border", "1px solid red");
  • * - выбирает все элементы, включая head и body.
    $("*").css("border", "1px solid red");
  • selector1, ..., selectorN - выбирает все элементы, переданные в селекторах.
    $("div, span, p.lok").css("border", "1px solid red");


Иерархические селекторы

  • parent child - выбирает все элементы, являющиеся потомками для определенного элемента предка.
    $("form input").css("border", "1px solid red");
  • parent > child - выбирает всех прямых наследников элемента-родителя.
    $("#content > *").css("border", "1px solid red");
  • Селекторы форм
    :input - выбираются все элементы input.
    :text - выбираются все элементы input типа text.
    :password - выбираются все элементы input типа password.
    :radio - выбираются все элементы input типа radio.
    :checkbox - выбираются все элементы input типа checkbox.
    :submit - выбираются все элементы input типа submit.
    :reset - выбираются все элементы input типа reset.
    :button - выбираются все элементы input типа button.
    :image - выбираются все элементы input типа image.
    :file - выбираются все элементы input типа file.
    :hidden - выбираются все элементы input типа hidden или просто скрытые.


Фильтры

  • :first - выбирает первый элемент соответствующего селектора.
    $("tr:first").css("font-style", "italic");
  • :last - выбирает последний элемент соответствующего селектора.
    $("tr:last").css("font-style", "italic");
  • :even - выбирает четные элементы, начиная с нуля.
  • :odd - выбирает нечетные элементы, начиная с нуля.
  • :eq(index) - выбирает элемент по его индексу (начиная с нуля).
    $("td:eq(2)").css("font-style", "italic");
  • :gt(index) - выбирает элементы с индексом больше указанного.
    $("td:gt(2)").css("font-style", "italic");
  • :lt(index) - выбирает элементы с индексом меньше указанного.
    $("td:lt(4)").css("font-style", "italic");


Фильтры видимости

  • :hidden - выбирает все скрытые элементы и элементы input, имеющие тип "hidden".
    $("div:hidden").show();
  • :visible - выбирает все видимые элементы.
    $("div:visible").css("color", "red");


Фильтры атрибутов

  • [attribute] - выбирает все элементы с указанным атрибутом.
    $("div[id]").css("color", "red");
  • [attribute=value] - выбирает элементы с атрибутом, значение которого соответствует указанному.
    $("input[name='Main']").css("color", "red");
  • [attribute^=value] - выбирает все элементы, соответствующий атрибут которых начинается с указанного значения.
    $("input[name^='news']").css("color", "red");
  • [selector1]...[selectorN] - выбирает все элементы, которые имеют указанные атрибуты и соответствующие значения.
    $("input[id][name*='news']").css("color", "red");

Фильтры форм

  • :enabled / :disabled - выбирает все элементы, имеющие активное (enabled) / запрещенное (disabled) состояние.
    $("input:enabled").css("color", "red");
  • :checked - выбирает все элементы, которые отмечены.
    $("input:checked").lenght;
  • :selected - выбирает все выбранные элементы. $("select option:selected").lenght;


Методы для работы с CSS-стилями

В jQuery имеется три категории методов: одни манипулируют с элементами, подходящими по шаблону; вторые возвращают значения элемента, а третьи изменяют сами элементы.

  • .css(name,value) - Добавление какому-либо элементу стиль
    $("div").css("border", "1px solid blue");
  • .css({properties}) - Если необходимо задать для элемента несколько CSS-правил, то лучше использовать эту конструкцию
    $("div").css({ border:"1px solid blue", fontWeight:"bolder", backgroundColor:"red" });
  • .addClass(class) - Добавление класса $("p:last").addClass("main");
  • .removeClass(class) - Удаление класса $("p:even").removeClass("main");
  • .toggleClass(class) - Удаление класса, если таковой присутствует, либо добавление класса, если таковой отсутсвует. $("p").toggleClass("main");
  • .height([value]) или .width([value]) - Задание или получение высоты/ширины элемента $("div").height();
    $("div").height(200);


Методы для работы с html

  • html(val) - добавляет html-код в выбранные элементы.
    $("div.sp").html("<span>Привет</span>");
  • .text() / .val() - возвращает текстовое содержимое элемента / значение элемента.
    $("p").text();
  • text(val) / val(val) - установит текст для элемента / значение для элемента.
    $("p").text("Привет!");
  • append(content) - добавляет content внутрь всех выбранных элементов ПОСЛЕ существующего контента.
    $("p").append("<b>Привет</b>");
  • appendTo(content) - обратная операция, добавляет В content все выбранные элементы.
    $("p").appendTo("#main");
    Иными словами:
    $(A).append(B) - добавит В в конец А.
    $(A).appendTo(B) - добавит А в конец В.
  • prepend(content) - добавляет content внутрь всех выбранных элементов ДО существующего контента.
    $("p").prepend("<b>Привет</b>");
  • prependTo(content) - обратная операция, добавляет В content все выбранные элементы.
    $("p").prependTo("#main"); Иными словами:
    $(A).prepend(B) - добавит В в начало А.
    $(A).prependTo(B) - добавит А в начало В.
  • wrap(html) - оборачивает каждый элемент в элемент обертку.
    $("p").wrap("<span></span>");
  • wrapAll(html) - оборачивает все элементы в ОДИН элемент обертку.
    $("p").wrapAll("<span></span>");
  • wrapInner(html) - оборачивает внутренне содержание каждого элемента.
    $("p").wrapInner("<em></em>");
  • replaceWith(content) - замещает одни элементы другими.
    $("span").replaceWith("<div></div>");
  • replaceAll(selector) - обратная операция, т.е. все selector-ы будут поменяны на элементы.
    $("span").replaceAll("<div></div>");
  • empty() - удаляет из элемента все узлы-потомки, включая текст.
    $("span").empty();
  • remove() - удаляет сами элементы.
    $("span").remove();
  • clone() - клонирует элементы.
    $("b").clone().append("div");


Методы для работы с атрибутами и фильтрами

  • attr(name) - обеспечивает доступ к значению указанного атрибута первого элемента в наборе.
    var a=$("i").attr("title");
    $("div").text(a);
  • attr(properties) - установит атрибуты во всех отобранных элементах.
    $("img").attr({src:"images/pict.gif", alt:"рисунок"});
  • attr(key,value) - установит значение (value) атрибута (key) для всех отобранных элементов.
    $("button").attr("disabled", "disabled");
  • removeAttr(name) - удалит указанный атрибут в всех элементах.
    $("img").removeAttr("alt");
  • hasClass(class) - возвращает истину (true), если указанный класс присутствует хотя бы в одном из элементов.
    if ($("p:last").hasClass("selected"))
    $(this).css("background","blue");
  • filter(expr) - ограничивает элементы, к которым следует что-либо применить.
    $("p").filter(".blue").css("background","blue");
  • prependTo(content) - обратная операция, добавляет В content все выбранные элементы.
    $("p").prependTo("#main");
  • not(expr) - обозначает элементы, к которым не следует что-либо применить.
    $("p").not(".blue").css("background","blue");
  • is(expr) - возвращает истину (true), если хотя бы один из элементов соответствует выражению.
    if ($(this).is(":last-child"))
    $("p").text("последний");
  • slice(start, end) - отбирает поднабор из набора элементов.
    $("p").slice(1,4).css("background","blue");


Методы по обработке событий

  • ready(fn) - назначает функцию, которая будет выполняться, когда документ готов к работе.
    $(document).ready(init);
  • bind(type, fn) - связывает обработчик события с самим событием.
    $(div).bind('click', init);
  • one(type, fn) - связывает обработчик события с самим событием, но выполняется он только один раз.
    $(div).one('click', init);
  • hover(over, out) - когда указатель мыши находится над объектом, срабатывает функция over, а когда указатель мыши выходит за объект, то функция out.
    $(document).ready(init);
    function init(){ $('p').hover(hOver, hOut);
    }
    function hOver(){
    $(this).css("background-color", "blue");
    }
    function hOut(){
    $(this).css("background-color", "white");
    }
  • toggle(fn1, fn2, ...fn) - переключатель между функциями. Щелчок по элементу вызывает функцию fn1, повторный щелчок - функцию fn2, третий щелчок - функцию fn3 и т.д.
    $('p').toggle(
    function(){
    $(this).css("background-color", "blue");
    },
    function(){
    $(this).css("background-color", "white");
    });
  • click(fn) - функция fn связывается с событием click.
    $('div').click( function(){$(this).css("background-color", "blue");} );


Визуальные эффекты

  • Методы видимости - hide(), show() и toggle()
    hide() - скрыть,
    show() - показать,
    toggle() - показать, если скрыты и скрыть, если видимы.
  • Методы видимости(продолжение)
    hide(speed, callback) - скрыть,
    show(speed, callback) - показать,
    toggle(speed, callback) - переключить (показать, если скрыты и наоборот), где:
    speed - скорость изменения высоты, ширины или свойства opacity (прозрачность) элемента. Может принимать три значения: slow (медленно), normal (нормально) или fast (быстро), а также значение в миллисекундах.
    callback - функция, которая будет выполняться после завершения анимации. Ее присутствие вовсе необязательно.
  • Методы свертывания - slideDown(), slideUp() и slideToggle()
    slideUp() - свернуть,
    slideDown() - развернуть,
    toggle() - развернуть, если скрыт и свернуть, если видим.
  • Методы свертывания(продолжение)
    slideUp(speed, callback) - свернуть,
    slideDown(speed, callback) - развернуть,
    slideToggle(speed, callback) - переключить (развернуть, если скрыты и наоборот), где:
    speed - скорость изменения высоты элемента. Может принимать три значения: slow (медленно), normal (нормально) или fast (быстро), а также значение в миллисекундах.
    callback - функция, которая будет выполняться после завершения анимации. Ее присутствие необязательно.
  • Методы исчезновения - fadeTo(), fadeOut() и fadeIn()
    fadeTo(speed, opacity, callback) - уменьшает свойство opacity (прозрачность) к заданному значению,
    fadeOut(speed, callback) - уменьшает свойство opacity (прозрачность),
    fadeIn(speed, callback) - увеличивает свойство opacity (прозрачность), где:
    speed - скорость изменения прозрачности элемента. Может принимать три значения: slow (медленно), normal (нормально) или fast (быстро), а также значение в миллисекундах.
    opacity - значение прозрачности, до которого оно будет уменьшено (число от 0 до 1).
    callback - функция, которая будет выполняться после завершения анимации. Ее присутствие необязательно.
  • Методы анимации - animate() и stop()
    animate(params, options) - анимирует стилевые свойства, где:
    params - атрибуты css, которые хотим анимировать ("width", "top", "border"...).
    options - свойства анимации (в том числе скорость).
    stop() - останавливает анимацию.
